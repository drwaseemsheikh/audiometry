\doxyparagraph*{Getting started}


\begin{DoxyEnumerate}
\item Use the Nu\+Get package manager to add a reference to Oxy\+Plot
\item Add a {\ttfamily Plot\+View} to your user interface
\item Create a {\ttfamily Plot\+Model} in your code
\item Bind the {\ttfamily Plot\+Model} to the {\ttfamily Model} property of your {\ttfamily Plot\+View}
\end{DoxyEnumerate}
